# mai_python

Телеграм-бот для рекомендаций мест отдыха и развлечений.

## Быстрый старт

1. Создайте `.env` по примеру `.env.example` и добавьте токен бота.
2. Установите зависимости:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. Запустите бота:

```bash
python bot.py
```

## Что уже есть

- Главное меню и базовая навигация.
- Подбор места по категориям, цене и рейтингу.
- Текстовый поиск и случайная рекомендация.
- Карточка места с избранным, лайками и отзывами (часть функций — заглушки).
- Пошаговое добавление нового места с отправкой на модерацию.
- Простая панель модератора (`/admin`) для approve/reject.

## Заглушки

- Геопоиск по координатам (кнопка "Рядом со мной").
- Лайки копятся, но пока не влияют на ранжирование.
- LLM-сводка отзывов заменена на простую эвристику.
- Отзывы показываются без ранжирования.

## Данные

Хранятся в папке `data/` в JSON-файлах. При первом запуске добавляются тестовые места.
